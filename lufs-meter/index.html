Implementation Notes

이 미터는 브라우저 탭 오디오 캡처와 Web Audio API 기반의 근사 구현입니다. YouTube iframe 오디오는 직접 연결할 수 없어서, 브라우저의 탭 오디오 공유 권한을 통해 들어온 출력 신호를 분석합니다. K-weighting은 RBJ biquad 기반의 high-pass와 high-shelf 필터로 근사했고, Integrated LUFS는 400ms block, absolute gate -70 LUFS, relative gate를 적용합니다.

True Peak는 정확한 oversampling이 필요한 영역이라 현재는 sample peak max로 표시합니다.
